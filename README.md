## 📦 Installation

### 1. **Cloner le projet**
```bash
git clone https://github.com/ton-repo/mini-bot-dev-shadow.git
cd mini-bot-dev-shadow
```

### 2. **Installer les dépendances**
```bash
npm install
```

### 3. **Lancer localement**
```bash
npm start
```

> Le bot sera accessible sur `http://localhost:3002`

---

## 🌐 Déployer sur Vercel

### Méthode 1 : Via le Dashboard Vercel

1. Créer un compte sur **vercel.com**
2. Connecter ton GitHub (ou importer le projet)
3. Cliquer sur **"Import Project"**
4. Sélectionner ce projet
5. Cliquer **"Deploy"** ✅

### Méthode 2 : Vercel CLI

```bash
npm install -g vercel
vercel login
vercel --prod
```

## 📱 Utilisation du Bot

### Commandes Disponibles

#### 🎮 **Fun**
- `.ping` - Réponse rapide
- `.hello` - Salutation
- `.joke` - Blague aléatoire
- `.dice` - Dé aléatoire
- `.coin` - Pile ou Face
- `.quote` - Citation inspirante
- `.love` - Taux d'amour
- `.dance` - Danse
- `.smile` - Sourire

#### 🧮 **Outils**
- `.calc 2+2` - Calcul
- `.time` - Heure actuelle
- `.date` - Date actuelle
- `.uptime` - Uptime du bot
- `.weather` - Météo
- `.translate` - Traduction
- `.qr` - Code QR

#### 👥 **Groupe**
- `.kick` - Expulser un membre
- `.ban` - Bannir un membre
- `.promote` - Promouvoir en admin
- `.demote` - Rétrograder
- `.mute` - Silence du groupe
- `.unmute` - Remise en service
- `.groupinfo` - Info du groupe
- `.members` - Liste des membres

#### 📝 **Texte**
- `.toupa texte` - Majuscules
- `.tolower texte` - Minuscules
- `.reverse texte` - Inverser
- `.repeat 3 texte` - Répéter

#### ⚙️ **Admin**
- `.restart` - Redémarrer
- `.stop` - Arrêter le bot
- `.owner` - Infos owner
- `.prefix` - Préfixe du bot

#### 📖 **Info**
- `.help` - Aide complète
- `.menu` - Menu principal
- `.about` - À propos du bot
