## 📱 UTILISATION

### Accéder au site de Pairing

```
https://ton-bot.vercel.app/
```

### Utiliser le bot

1. Entre ton numéro WhatsApp
2. Récupère le code de pairing
3. Dans WhatsApp : Paramètres > Appareils connectés > Connecter
4. Entre le code
5. **C'est prêt !** 🎉

